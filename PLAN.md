## 1. 프로젝트 목표
Python으로 콘솔 기반 할 일 관리 프로그램을 만든다.
사용자는 터미널에서 메뉴를 선택하여 할 일을 추가, 조회, 완료 처리, 삭제할 수 있다.--
## 2. 구현할 기능
1. 할 일 추가
2. 할 일 목록 조회
3. 할 일 완료 처리
4. 할 일 삭제
5. 프로그램 종료--
## 3. 구현하지 않을 기능
이번 실습에서는 다음 기능은 구현하지 않는다.- 로그인 기능- 웹 화면- 데이터베이스 연동- 파일 저장 기능- 복잡한 예외 처리--
## 4. 파일 구조
예상 파일 구조는 다음과 같다.
todo-ai-practice/
├── PLAN.md
└── main.py--
## 5. 프로그램 동작 방식
프로그램을 실행하면 다음 메뉴가 출력된다.
==== 할 일 관리 프로그램 ====
1. 할 일 추가
2. 할 일 목록 보기
3. 할 일 완료 처리
4. 할 일 삭제
5. 종료
메뉴를 선택하세요:
사용자는 번호를 입력하여 원하는 기능을 실행한다.--
## 6. 작업 순서- [ ] 1단계: 기본 메뉴 출력 만들기- [ ] 2단계: 할 일 추가 기능 만들기- [ ] 3단계: 할 일 목록 조회 기능 만들기- [ ] 4단계: 완료 처리 기능 만들기- [ ] 5단계: 삭제 기능 만들기- [ ] 6단계: 전체 실행 테스트하기--
## 7. 코드 작성 규칙- 초보자가 이해하기 쉬운 Python 코드로 작성한다.- 함수 단위로 코드를 나눈다.
4
PLAN.md 기반 AI 코딩 실습 자료- 외부 라이브러리는 사용하지 않는다.- 각 함수에는 간단한 주석을 작성한다.--
## 8. 테스트 방법
1. 프로그램을 실행한다.
2. 할 일을 2개 추가한다.
3. 할 일 목록을 조회한다.
4. 첫 번째 할 일을 완료 처리한다.
5. 두 번째 할 일을 삭제한다.
6. 다시 목록을 조회한다.
7. 프로그램을 종료한다.--
## 9. 완료 기준- 프로그램이 에러 없이 실행된다.- 할 일 추가, 조회, 완료, 삭제가 가능하다.- 메뉴가 반복해서 출력된다.- 종료 메뉴를 선택하면 프로그램이 종료된다.
